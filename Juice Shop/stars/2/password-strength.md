
# Password Strength
## Log in with the administrator's user credentials without previously changing them or applying SQL Injection.

El desafio consiste en acceder a la web como admin, para lo cual habra que crackear la clave para poder acceder.

* Interceptar peticion de login con burpsuite.
* Crear/usar diccionario de claves.
* Usar el intruder de burpsuite para hacer un ataque de fuerza buruta, consiguiendo asi la clave.
