# Web3 Sandbox
## Find an accidentally deployed code sandbox for writing smart contracts on the fly.

Buscando en el codigo de el archivo `main.js` se puede encontrar una ruta llamada `web3-sandbox`

* Acceder desde el navegador a `herramientas de desarrollador` > `debuger`
* Buscar en el main.js la palabra `web3-sandbox`
* Ir a la ruta http://172.17.0.3:3000/#/web3-sandbox
