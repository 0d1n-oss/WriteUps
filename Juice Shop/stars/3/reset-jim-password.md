# Reset Jim's Password
## Reset Jim's password via the Forgot Password mechanism with the original answer to his security question.

El reto consiste en cambiar la clave de el usuario jim usando el formulario de cambio de clave.
Dependemos de la pregunta de seguridad para cambiar la clave, asi que habra que investigar sobre jim.

Resulta que el usuario es `James T. Kirk` lo cual simplifica el asunto, ya que es un personaje conocido.

La pregunta de seguridad es sobre su hermano, el cual es `George Samuel Kirk`.
