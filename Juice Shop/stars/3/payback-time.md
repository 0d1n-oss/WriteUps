# Payback Time
## Place an order that makes you rich.

Para resolver el reto hay que agregar un prodicto al carro de compras.

* Agregar un producto al carro de compras.
* Interceptar con burpsuite mientras se aumenta la cantidad del producto.
* Desde la peticion de burpsuite hay que modificar el parametro `quantity` para cambiar el valor a un numero negativo.
* Ya con esto hecho solo queda completar los pasos para comprar el producto.
